# chat-app-
This is a Real-Time Chat Application built using Node.js, Express, and Socket.IO that enables users to communicate instantly over the web. The application uses WebSocket-based communication to deliver messages in real-time without requiring page refreshes.

It supports multiple users and demonstrates core concepts of full-stack development, including client-server architecture, event-driven programming, and real-time data exchange.
